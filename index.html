Privacy Policy

Last Updated: June 11, 2026

Your privacy is fundamentally important to us. This Privacy Policy details the types of information Jeeves collects, how it is processed, and stored.

1. Information We Collect

  • Discord Meta Data: User IDs, username strings, server IDs, and channel IDs to route commands correctly.
  • User Interaction Input: Text commands and configurations submitted directly to the bot.
  • Long-Term Memory Data: Personal facts explicitly provided by users to build an ongoing personalized assistant pipeline, stored securely using backend vector databases.

2. How We Use Information

Collected information is used strictly to operate and enhance Jeeves' core functionalities: to execute user-triggered automation routines and to retrieve conversational contextual markers via vector embeddings for personalized responses.

3. Data Retention and Storage

Memory banking records remain stored securely in isolated databases until an account deletion is requested. Transient cache data is automatically purged inside short-term system windows.

4. Data Erasure and Control

You maintain full authority over your data. If you wish to purge saved personalization records stored within Jeeves' vector memory banks, you can issue the internal cleanup commands directly inside Discord.
