Privacy Policy

Last Updated: June 11, 2026

Your privacy is fundamentally important to us. This Privacy Policy details the types of information Jeeves collects, how it is processed, and stored.

1. Information We Collect

  • Discord Meta Data: User IDs, username strings, server IDs, and channel IDs to route commands correctly.
  • User Interaction Input: Text commands and configurations submitted directly to the bot.
  • Long-Term Memory Data: Personal facts explicitly provided by users to build an ongoing personalized assistant pipeline, stored securely using backend vector databases.

2. How We Use Information

Collected information is used strictly to operate and enhance Jeeves' core functionalities: to execute user-triggered automation routines and to retrieve conversational contextual markers via vector embeddings for personalized responses. We do not sell or lease your personal data to third parties.

3. AI Processing and Third-Party Infrastructure

To provide advanced conversational capabilities, text strings submitted to Jeeves are processed through secure Large Language Model (LLM) APIs and maintained via protected cloud database providers. These sub-processors are prohibited from using your data to train public baseline models and must handle it in compliance with standard data protection policies.

4. Data Retention and Storage

Memory banking records remain stored securely in isolated databases until an account deletion is requested. Transient cache data and localized system states are automatically purged inside short-term system windows.

5. Data Erasure and Control

You maintain full authority over your data. If you wish to purge saved personalization records stored within Jeeves' vector memory banks, you can issue the internal cleanup commands directly inside Discord. Upon trigger, backend tasks will completely delete your associated vector coordinates and contextual data payloads from our databases.

6. Security Measures

We employ standard operational encryption protocols to protect your data during transmission and while stored at rest. Access tokens, credentials, and user data vectors are tightly isolated to prevent external exposure.

7. Children's Privacy

Jeeves is not directed at or intended for children under the age of 13. We do not knowingly collect personal information from children. If we discover that a child under 13 has provided data, we will immediately execute background tasks to purge it.
